## Execution

Firtly, it is necessary to have installed NPM, Node.js, Gulp, Browserify and Bower.

**(1)** Install Node Package Manager dependencies.

`npm install`

**(2)** Start Gulp.

`gulp`

**(3)** Start the server.

`npm watch start`

**(4)** Access the web page through the browser.

`http://localhost:3000`
